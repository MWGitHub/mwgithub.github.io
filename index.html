Code Splat

A crowd sourced code annotation site.

New

Code Splat is a web application for annotating code and text. It was inspired by RapGenius and built using Ruby on Rails and React.js.

Code is displayed with highlights and correct indentations through CodeMirror integration for the viewer and editor. In order to highlight text for annotations, the character index had to be calculated from the line and character due to the differences between the library's format and the way the data is stored on the backend. The calculated index is stored and converted back when the text is rendered in order to highlight spots that are annotated.
